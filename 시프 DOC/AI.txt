
AI가 필요한 순간
  1. 주인공이 명령을 결정한 순간 필드의 상황을 보고 명령 선택
  2. 적의 포켓몬이 죽어서 상황에 알맞는 포켓몬 선택


명령 AI
  if 주인공의 포켓몬에게 불리한 경우 (&& 남아있는 포켓몬수 > 1 && 포켓몬 리스트중 상성상 우위에 있는 포켓몬이나 상성이 없는 포켓몬이 있을때) 
    -> 포켓몬 교체  // 우선순위 (1. 상성상 우위에 있는 포켓몬 2. 상성이 없는 포켓몬)
  else if 피가 1/3 이하인 경우 (&& 남아있는 상처약수 > 0)
    -> 상처약 사용
  else             // 공격기 선택의 우선순위
    -> 공격기 사용 (1. 상대한테 2배 데미지 줄수있는 기술 2. 상대한테 1배 데미지 줄수있는 기술 3.위력이 가장 큰 기술)

포켓몬 채우기 AI
  주인공의 필드 포켓몬을 보고 유리하거나 동등한 포켓몬, 없을경우 랜덤으로 return;
